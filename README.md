## Configs
- Nix (OS): `/nix`
- Hyprland (WM): `/.config/hypr`
- Kitty (Terminal): `/.config/kitty`
- Waybar (Topbar): `/.config/waybar`
- Bash (shell): `.bashrc`
- Git (SCM): `.gitconfig`
